# Smart Job Application Tracker

A simple Python-based application designed to help users track and manage their job applications efficiently.

## Features
- Add new job application details
- View all submitted job applications
- Update the status of applications (Applied, Interview, Rejected, Offer)
- Persistent data storage using a CSV file

## How to Run
1. Open the project folder in a terminal  
2. Run the following command:

```bash
python job_tracker.py
